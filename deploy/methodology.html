Core Methodology

1. Semantic Embedding with SPECTER2

What: Transform research papers into high-dimensional semantic vectors using a transformer model specifically trained on scientific literature.

Why: SPECTER2 understands scientific language better than general-purpose models, capturing domain-specific relationships and terminology.

Technical Details: 768-dimensional embeddings, cosine similarity, batch processing with GPU acceleration when available.

Reference: SPECTER2: Better Scientific Paper Embeddings

2. Graph-Based Clustering with Leiden Algorithm

What: Build k-nearest neighbor graphs from embeddings and apply community detection to find research clusters.

Why: Leiden algorithm provides superior clustering quality compared to traditional methods like k-means, especially for network data.

Technical Details: k=15 nearest neighbors, cosine similarity metric, resolution parameter optimization, modularity maximization.

Reference: From Louvain to Leiden: guaranteeing well-connected communities

3. Temporal Burst Detection

What: Identify periods of unusual research activity using statistical burst detection methods.

Why: Reveals emerging trends, foundational periods, and the temporal evolution of research streams.

Technical Details: Kleinberg burst detection, z-score analysis, prevalence tracking over time.

Reference: Bursty and Hierarchical Structure in Streams

4. Automated Dialog Card Generation

What: Generate comprehensive research stream summaries using AI-powered analysis and natural language generation.

Why: Provides human-readable insights that combine quantitative analysis with qualitative understanding.

Technical Details: BERTopic for theme extraction, TF-IDF for keyword analysis, automated paper categorization.

Reference: BERTopic: Neural Topic Modeling with BERT

Technical Pipeline

Data Preprocessing & Parsing

Parse BibTeX files, clean LaTeX formatting, normalize author names and journal titles, extract DOIs and abstracts.

  • Handle multiple text encodings (UTF-8, Latin-1)
  • Clean LaTeX accents and special characters
  • Parse complex author name formats
  • Create analysis-ready text fields (title + abstract)

Semantic Embedding Generation

Transform papers into high-dimensional semantic vectors using SPECTER2 transformer model.

  • Load pre-trained SPECTER2 model from HuggingFace
  • Batch processing with adaptive batch sizes
  • GPU acceleration when available
  • L2 normalization for cosine similarity

Graph Construction & Clustering

Build similarity graphs and apply advanced community detection algorithms.

  • Construct k-NN graphs with cosine similarity
  • Apply Leiden algorithm with resolution optimization
  • Evaluate clustering quality with silhouette scores
  • Fallback to HDBSCAN or k-means if needed

Temporal & Citation Analysis

Analyze temporal patterns and enrich with citation data from OpenAlex API.

  • Compute cluster prevalence over time
  • Detect research bursts using statistical methods
  • Enrich with citation counts and reference networks
  • Identify foundational and emerging papers

Report Generation & Visualization

Generate comprehensive dialog cards and interactive visualizations.

  • Extract research themes with BERTopic
  • Identify key papers (most cited, central, foundational)
  • Generate natural language descriptions
  • Create interactive HTML dashboards

Quality Assurance & Validation

Clustering Quality Metrics

  • Silhouette Score: Measures how well papers fit their assigned clusters
  • Modularity: Evaluates the quality of network community structure
  • Stability Analysis: Tests clustering consistency across parameter variations
  • Manual Inspection: Expert review of cluster coherence and interpretability

Embedding Quality Validation

  • Semantic Coherence: Similar papers should have similar embeddings
  • Dimensionality: 768-dimensional SPECTER2 vectors capture rich semantic information
  • Normalization: L2-normalized vectors ensure fair cosine similarity comparisons
  • Coverage: All papers with sufficient text content are embedded

Temporal Analysis Validation

  • Burst Significance: Statistical testing of prevalence changes
  • Temporal Consistency: Research streams should show coherent evolution
  • Historical Accuracy: Foundational papers should precede derivative work
  • Trend Validation: Emerging streams should align with domain knowledge
